xMenu3

xMenu3 is my latest menu using the X library. It can be used alone or as a component of Hawk. For now, I have more xMenu3 documentation at the Hawk site than here. xMenu3 is part of the Hawk download.

Turn javascript off and reload this page to see its downgrade mode.

Discuss xMenu3 at the SitePoint Javascript Forum.

CSS

A menu's appearance is defined almost completely with CSS.




HTML


There are minimal HTML requirements.





Javascript


Just instantiate a menu object.


this.m1 = new xMenu('xMenu1',    // element id
  'horizontal', 2, -6,           // mnuType, verOfs, hrzOfs
  -12, -16, -20, -4,             // lbl selection area clipping (t,r,b,l)
  -32, null, null, null,         // box selection area clipping
  'myHMBarLbl', 'myHMBarLblHvr', // barLblOutStyle, barLblOvrStyle
  'myHMBarLblHvrClosed',         // barLblOvrClosedStyle
  'myMLbl', 'myMLblHvr',         // lblOutStyle, lblOvrStyle
  'myHMBar', 'myMBox');          // barStyle, boxStyle



Then size and position the menu container. Then load, paint, and show it.


xResizeTo(this.m1.ele, w, this.m1H);        // menu 1
xMoveTo(this.m1.ele, x, xPageY(this.h) + xHeight(this.h)); 
this.m1.load();
this.m1.paint();
xShow(this.m1.ele);
